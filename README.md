# 🔄 Flatten Iterator & Generator

## 📌 Описание

Проект реализует:

- Итератор для плоского обхода списка списков
- Генератор для плоского обхода
- Поддержку любой глубины вложенности (рекурсивно)

---

## ⚙️ Функциональность

### ✅ Итератор (1 уровень)
Класс `FlatIterator` обрабатывает список списков

### ✅ Генератор (1 уровень)
Функция `flat_generator`

### ✅ Итератор (любая вложенность)
Класс `FlatIteratorRecursive`

### ✅ Генератор (любая вложенность)
Функция `flat_generator_recursive`
